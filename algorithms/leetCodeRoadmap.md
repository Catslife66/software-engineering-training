- Merge Intervals
- Insert Interval

## Module C — Monotonic Stack

New question:

What previous element is still useful?

Examples:

- Daily Temperatures
- Next Greater Element
- Stock Span

## Module D — Binary Search on Answers

A subtle but powerful shift:

You’re not searching an array.

You’re searching the answer space.

Examples:

- Koko Eating Bananas
- Capacity to Ship Packages
- Split Array Largest Sum

## Module E — Heap / Priority Queue

Engineering question:

How do I always retrieve the “best” element efficiently?

Examples:

- Top K Frequent Elements
- K Closest Points
- Merge K Sorted Lists
