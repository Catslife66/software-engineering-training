# Roadmap

## Stage 1 — Stabilize Backtracking Core (Current)

Continue drilling:

- Subsets
- Combination Sum
- Permutations
- Generate Parentheses

Goal:

```
make choose → recurse → undo automatic
```

## Stage 2 — Grid DFS

Next major phase.

Examples:

- number of islands
- flood fill
- word search
- maze/path problems

New concepts:

- visited marking
- coordinate movement
- boundary checks

This is VERY important interview territory.

## Stage 3 — Advanced Backtracking

Examples:

- palindrome partitioning
- sudoku solver
- N queens
- word break DFS

These require:

```
pruning + state optimization
```

## Stage 4 — Graph DFS/BFS

Then transition to:

- graph traversal
- cycle detection
- connected components
- topological sort

## Stage 5 — Dynamic Programming

ONLY after recursion becomes comfortable.

Because DP is:

```
optimized recursion
```

without recursion intuition, DP becomes memorization hell.
