## Your Review Plan (2-week cycle)

1. Complexity & Thinking

   Goal:
   - Understand Big-O deeply
   - Recognize patterns of time complexity

   Practice:
   - Compare O(n), O(n²), O(log n)
   - Trace loops manually

2. Arrays (Hashing + Two Pointers)

   Focus
   - Set vs Map
   - Two Sum (both versions)
   - Remove duplicates / uniqueness

   Practice
   - duplicate detection
   - two sum
   - remove duplicates (order preserved)

   Key question to ask yourself:
   - What information do I need to store?

3. Sliding Window

   Focus
   - fixed-size window
   - variable-size window

   Practice
   - max sum subarray (k)
   - smallest subarray ≥ S
   - longest substring without repeat

   Key question
   - When do I expand? When do I shrink?

4. Stack & Recursion

   Focus
   - valid parentheses
   - recursion mindset
   - base case vs recursive step

   Practice
   - valid parentheses
   - factorial / sum recursion
   - trace recursion tree

   Key question
   - What is the smaller problem?

5. Trees

   Focus
   - preorder / inorder / postorder
   - recursive tree thinking

   Practice
   - traversal (write by hand)
   - height of tree
   - simple tree problems

   Key question
   - Do I need children first or not?

6. Graphs (BFS / DFS)

   Focus
   - BFS vs DFS
   - islands
   - shortest path

   Practice
   - number of islands
   - shortest path grid
   - simple BFS traversal

   Key question
   - Is this shortest path or just exploration?

7. Dynamic Programming

   Focus
   - 1D DP (stairs, house robber)
   - 2D DP (grid)

   Practice
   - climbing stairs
   - house robber
   - min path sum
   - unique paths

   Key question
   - What is my state?
   - What is my transition?
