## Your Review Plan (2-week cycle)

1. Complexity & Thinking

   Goal:
   - Understand Big-O deeply
   - Recognize patterns of time complexity

   Practice:
   - Compare O(n), O(n²), O(log n)
   - Trace loops manually

2. Arrays & Hash Maps

   Patterns:
   - duplicate detection
   - frequency counting
   - lookup optimization

   Key idea:
   - trade space for time

3. Two Pointers & Sliding Window

   Patterns:
   - sorted array → two pointers
   - subarray problems → sliding window

   Key signals:
   - "sorted"
   - "subarray"
   - "continuous"

4. Stack & Recursion
   - Stack:
     - parentheses
     - undo / reverse logic
   - Recursion:
     - base case
     - call stack
     - tree thinking

5. Graphs (BFS / DFS)

   Patterns:
   - islands (connected components)
   - shortest path (BFS)
   - multi-source BFS

   Key questions:
   - what are nodes?
   - what are neighbors?

6. Dynamic Programming

   Sub-patterns:
   - Fibonacci / stairs
   - House robber
   - Grid DP
   - Obstacle handling

   Core framework:
   - State → Transition → Base case → Operation
