## The 8 Core Patterns

1. Hash Map (Counting / Lookup)

Used when:

- counting
- finding duplicates
- fast lookup

2. Two Pointers

Used when:

- sorted array
- pairs
- palindrome

3. Sliding Window

Used when:

- subarray / substring
- continuous segment
- max/min window

4. Binary Search

Used when:

- sorted data
- search space shrinking

5. Recursion

Used when:

- tree
- divide problem
- repeating structure

6. Backtracking

Used when:

- all combinations
- permutations
- decision tree

7. Tree Traversal (DFS / BFS)

Used when:

- hierarchical data
- exploring nodes

8. Dynamic Programming

Used when:

- overlapping subproblems
- optimal decisions

## The Thinking Framework

1. Identify input type
2. Identify goal
3. Map to pattern
4. Plan solution
5. Code
